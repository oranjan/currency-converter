# **Currency Converter **

A simple, fast, and clean currency converter built with **React + Vite**. Fetches real-time exchange rates using the Frankfurter API and lets users convert any currency instantly.

## **Features**

- Convert between any two global currencies
- Real-time exchange rates via Frankfurter API
- Clean UI with swapped currency toggle
- Input validation + error handling
- Built using modern React (hooks, components)

## **Tech Stack**

- **React**
- **Vite**
- **Tailwind CSS**
- **Frankfurter API** (for rates)

## **Installation**

Clone the repo:

```bash
git clone https://github.com/oranjan/currency-converter.git
cd currency-converter
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
